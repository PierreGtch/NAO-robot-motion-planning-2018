Changes since 1.12¶

  • There are now 4 packages instead of one, each containing the correct pre-compiled version of the jnaoqi library.
  • Examples have been cleaned up.
  • The native library is no longer called JNaoQi, and is now called jnaoqi.
// In 1.12

static {
  System.LoadLibrary("JNaoQi");
 }

 // In 1.14:
 static {
   System.LoadLibrary("jnaoqi");
 }
