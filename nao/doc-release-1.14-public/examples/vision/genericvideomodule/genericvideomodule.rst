Generic video module
====================

:ref:`\<\< return to examples index <example-index>`

Overview
--------

**ALGenericVideoModule** is an example showing how to use various functions
from :ref:`ALVideoDevice`.

This is a **local module**, it must be cross-compiled and sent to the robot.
