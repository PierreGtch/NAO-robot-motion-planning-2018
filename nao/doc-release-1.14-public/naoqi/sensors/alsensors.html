ALSensors¶

NAOqi Sensors - Overview | API

See also

  • Contact Sensors Hardware

What it does¶

The ALSensor module is responsible for raising events corresponding to NAO’s bumper, tactile head, taclile hands and chest button hardware.

For further details, see section: ALSensors - List of events.

How it works¶

ALSensor module retrives sensors data in ALMemory, processes it and raises events accordind to the situation.
