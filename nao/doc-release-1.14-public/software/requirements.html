Choregraphe suite requirements¶

Hardware¶

A computer connected to a WiFi access point, with:

  • 1.5 GHz CPU
  • 512 MB RAM
  • a certified OpenGL graphics card

Supported Operating Systems¶

OS Version
Linux Ubuntu 10.04 LTS (Lucid) to 12.04 LTS (Precise) and later
Windows

Microsoft Windows XP Service Pack 3,

Microsoft Windows 7
Mac

Mac OS X version 10.7.1 Lion

Mac OS X version 10.8.3 Mountain Lion
